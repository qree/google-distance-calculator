== Description ==

MK Google Distance Calculator: You can use the MK Google Distance Calculator Plugin to find out the distance between two points anywhere on the earth. In other words, the distance between A and B. This plugin will also provide you the approximate time taken to travel between the two locations and the Driving Direction to the destination location.

== Installation ==

1. Upload the plugin to your /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Under the plugins menu in the admin area you will see MK-GDC Options page. Go to this page and set the parameters according to your requirements.

4. Now place the [MK-GDC] shortcode on the post/page where you want the MK Google Distance Calculator to appear.
