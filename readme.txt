== Description ==

MK Google Distance Calculator: You can use the MK Google Distance Calculator Plugin to find out the distance between two points anywhere on the earth. In other words, the distance between A and B. This plugin will also provide you the approximate time taken to travel between the two locations and the Driving Direction to the destination location.

== Installation ==

1. Upload the plugin to your /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Now place the [MK-GDC latitude="default map latitude" longitude="default map longitude"] shortcode on the post/page where you want the MK Google Distance Calculator to appear. The first parameter will be the latitude and the second will be the longitude of the default Google Map which will appear when the page first loads. For example your short tag may look like this [MK-GDC latitude="43.6525" longitude="-79.3816667"].
